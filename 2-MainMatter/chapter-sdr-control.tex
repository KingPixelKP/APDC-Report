\chapter{SDR Control}
\label{cha:sdr-control}

This chapter describes the software defined radio (SDR) control system used in this internship. The SDR control system is responsible for managing the communication between the SDR hardware and the software applications that utilize it. It provides an interface for configuring the SDR parameters, such as frequency, rf-control, and sampling rate, as well as handling data transmission and reception.
