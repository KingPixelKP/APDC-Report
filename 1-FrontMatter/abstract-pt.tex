Este relatório descreve o trabalho realizado durante o estágio de \gls{CDA}
realizado na \gls{GMV}, em Lisboa, Portugal, como parte do curso de
Licenciatura em Engenharia Informática na \gls{FCT}. O estágio decorreu entre
11 de março e 17 de julho de 2026, sob a supervisão da \gls{GMV}. O trabalho
desenvolvido durante este estágio centrou-se no design e implementação de um
sistema de software em C++ para o controlo programático de um gerador de sinais
\gls{GNSS} e \gls{SBAS}. Isto envolveu a interface com hardware profissional \gls{SDR}
através de uma API de fornecedor, o design de uma arquitetura cliente-servidor
baseada em gRPC para expor esse controlo remotamente, e a implementação de uma
camada de comunicação serial sobre \gls{RS232} para integração com equipamentos
externos. O relatório está organizado para refletir a progressão natural do
trabalho.
